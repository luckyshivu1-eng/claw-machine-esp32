# claw-machine-esp32

ODT Midterm assignment. This project implements a claw machine hook controlled by two servo motors and push buttons using an ESP32.

## Features
- Dual servo control using buttons  
- Edge-detected button input  
- NeoPixel status indicator  

## Hardware Used
- ESP32  
- 2 Servo motors  
- 4 Push buttons  
- NeoPixel ring (16 LEDs)
